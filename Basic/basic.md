# 기본 개념
- 객체(Object)
    - 소프트웨어 세계에 구현할 대상
    - 정의하는 것

- 메서드(Method)
    - 행동, 동작하는 것
    - 함수로고 할수도 있음

- 클래스(Class)
    - 객체를 구현하기 위한 설계도
    - 메서드, 변수등의 집합

- 인스턴스(Instance)
    - 설계도를 바탕으로 소프트웨어 세계에 구현된 구체적인 실체(객체를 구체화한 것)
    - 실체화된 인스턴스는 메모리에 할당
